# WorkSite Frontend Template

React + TypeScript + Vite starter for the WorkSite project.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS v4
- React Router
- Axios
- Zustand
- TanStack Query
- React Hook Form
- Lucide React

## Start

```bash
npm install
cp .env.example .env
npm run dev
```

Backend default:

```env
VITE_API_URL=http://localhost:8080/api
```

The API client uses `withCredentials: true` for HTTP-only JWT cookies.
